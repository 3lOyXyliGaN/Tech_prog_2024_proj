Detailed Description

Класс клиента

Класс реализует логику работы клиента с сервером

Warning
Перед использованием убедитесь, что сервер запущен на адресе 127.0.0.1:33333
Класс реализован как синглтон, не пытайтесь создать экземпляр напрямую
